2026-03-26

  1. Download ATAC-seq peak counts in the hippocampus upon stress (subset of the original data, already in SummarizedExperiment format).
  2. Using this object, run 2 differential analyses, respectively:
  • comparing stressed (denoted ‘FSS’ – forced swim stress) and control (CTRL) animals
  • comparing male and female animals (No need to worry about normalization here, you can just use the default TMM)
  1. For each analysis, report the top 10 most significant regions, and plot a heatmap of their normalized counts across the samples.

stressed (FSS) vs control (CTRL)

# DGEList and normalise
dds_stress <- edgeR::DGEList(assay(se), group = se$condition)
dds_stress  <- edgeR::calcNormFactors(dds_stress)   # default TMM
dds_stress  <- edgeR::estimateDisp(dds_stress)
## Using classic mode.
# Differential analysis 
et_stress <- as.data.frame(topTags(exactTest(dds_stress, pair = c("CTRL", "FSS")), n = Inf))
# volcano plot
ggplot(et_stress, aes(logFC, -log10(FDR))) +
  geom_point(alpha = 0.5) +
  ggtitle("FSS vs CTRL – volcano plot")

# stress comparison has only 1 region passing FDR < 0.05 –> region #15757 - region show increased chromatin accessibility after stress

# top 10 most significant regions 
top10_idx_stress <- as.integer(head(row.names(et_stress), 10))
se_top10_stress  <- se[top10_idx_stress, ]
rownames(se_top10_stress) <- as.character(seq_len(10))
assay(se_top10_stress, "logcpm", withDimnames=FALSE) <- log1p(cpm(dds_stress))[top10_idx_stress, ]
